НАВЫКИ И УМЕНИЯ

Что я использую в работе

HTML5

Создает скелет вашего сайта или приложения, а пятая версия позволяет создать оптимизированную SEO структуру вашего продукта.

CSS3

Этот язык стилей позволяет создать абсолютно любой внешний вид вашего сайта или приложения. Все ограничено только вашей фантазией!

Java Script

Этот язык программирования позволяет анимировать что угодно: слайдеры, окна, подсказки, вкладки, получение данных с сервера и многое другое.

Jquery

Библиотека jquery для ускорения разработки. Без необходимости, интегрировать ее в проект не буду, но навык работы с ней присутствует.

React

Эта библиотека позволяет создавать веб-приложения. Можно создать максимально интерактивный продукт именно для ваших целей

Redux

Это библиотека JavaScript с открытым исходным кодом для управления состоянием приложения. Чаще всего он используется с такими библиотеками, как React, для создания пользовательских интерфейсов.

WordPress

Это бесплатная система управления контентом (CMS) с открытым исходным кодом, написанная на PHP и связанная с базой данных MySQL или MariaDB.

Tilda

Блокчный конструктор сайтов. Он позволяет создавать веб-сайты, интернет-магазины, целевые страницы, блоги и электронные рассылки.
