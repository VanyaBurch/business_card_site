НАВИЧКИ

Що я використовую в роботі

HTML5

Створює скелет вашого сайту або програми, а п’ята версія дозволяє створити SEO-оптимізовану структуру вашого продукту

CSS3

Ця мова стилів дозволяє створювати абсолютно будь-який вигляд вашого веб-сайту чи програми. Все обмежується лише вашою фантазією!

Java Script

Ця мова програмування дозволяє анімувати що завгодно: повзунки, вікна, підказки, вкладки, отримання даних із сервера та багато іншого

Jquery

Бібліотека Jquery для пришвидшення розвитку. Без необхідності, інтегрувати в проект не буду, але навички роботи з нею присутні

React

Ця бібліотека дозволяє створювати веб-додатки. Можна створити найбільш інтерактивний продукт спеціально для ваших цілей

Redux

Це бібліотека JavaScript з відкритим кодом для управління станом програми. Найчастіше використовується з такими бібліотеками, як React, для побудови користувальницьких інтерфейсі

WordPress

Це безкоштовна система управління вмістом із відкритим кодом (CMS), написана на PHP і поєднана з базою даних MySQL або MariaDB.

Tilda

Блочний конструктор сайтів. Дозволяє створювати веб-сайти, інтернет-магазини, цільові сторінки, блоги та електронну розсилку.
